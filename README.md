### A Community-Driven YouTube Trending Page

This project is a website aimed at recreating the original, universal YouTube Trending Page, which was officially **discontinued in July 2025**.

The goal is to provide a single, non-personalized feed of videos that are currently gaining popularity across the platform, independent of individual user history.
